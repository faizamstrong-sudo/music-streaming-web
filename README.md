# FASHIN Play - Music Streaming Web Application

Modern music streaming web application dengan design batik Indonesia tradisional.

## Features
- Real-time music streaming
- Playlist management
- Dark/Light mode
- Responsive design
- Batik Jawa design aesthetic
- Personal greeting system

## Quick Start

### Prerequisites
- Node.js v20+
- Python 3+

### Installation & Running

1. Clone repository
2. Setup Backend:
   ```bash
   cd backend
   npm install
   npm start
   ```
3. Setup Frontend (terminal baru):
   ```bash
   cd frontend
   python -m http.server 8000
   ```
4. Open: http://localhost:8000
